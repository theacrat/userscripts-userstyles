# Userscripts

## [Reddit Image Proxy](userscripts/reddit-image-proxy.user.js)

Redirect Reddit's image viewer to a proxy.

## [Reddit Open in Apollo](userscripts/reddit-open-in-apollo.user.js)

Open Reddit links in Apollo.

## [Reddit Redesign Optout](userscripts/reddit-redesign-optout.user.js)

Default to old Reddit through the redesign_optout cookie.

## [Reddit Uncollapse](userscripts/reddit-uncollapse.user.js)

Expand auto-collapsed comments on old Reddit.

## [Reddit Undelete](userscripts/reddit-undelete.user.js)

Adds undelete button to posts and comments on old Reddit.

## [X Open in iOS App](userscripts/x-open-in-ios-app.user.js)

Open X/Twitter links in the iOS app (allows for tweaked apps).

## [YouTube Open in iOS App](userscripts/youtube-open-in-ios-app.user.js)

Open YouTube links in the iOS app (allows for tweaked apps).
